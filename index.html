Zero Search Engine

This is a demo of the Zero Search Engine. Please note that it is a work-in-progress, so expect to experience occasional errors and malfunctions. And please excuse the extremely bad-looking website; it will be greatly improved and redesigned in the future.

Zero is being developed by BrightSky OSSDO.

Known Problems

  • Only US-based websites have been crawled. This makes the search engine quite incomplete.
  • The search algorithm is still in development. You may occasionally see results that don't match your search query at all (although this is a rare occurrence).
  • Due to the way web crawling works, you may sometimes see a search result that is simply another search engine containing your query. But again, this is quite rare.
  • Bugs and glitches sometimes occur when parsing the metadata of a search result.
  • Attempting to include "?search=" in your search query will cause issues because it is a phrase reserved by Zero for parsing your search.
  • The page might break if you try to search for a really long word without spaces.
  • Search queries are limited to 200 characters.

Thank you for testing out Zero!
